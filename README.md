# ASCII-Art-Software
A simple ASCII Art Software built with Pygame.  

### CONTROLS:
**LMB** -> Paint  
**RMB** -> Erase  
**SCROLL** -> Zoom  
**MMB** or **LMB + SPACE** -> Move  
**UP and DOWN** -> Change character  
**ANY KEY** -> Set character to key  
**TAB** -> Toggle grid  
**CTRL + R** -> Clear canvas  
**CTRL + S** -> Save art  
**ESC** -> Quit  

### REQUIREMENTS:
Python 3.11+  
Pygame 2.3.1+  

Run `main.py` to start the software!
