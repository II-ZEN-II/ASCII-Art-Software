# ASCII-Art-Software
A simple ASCII Art Software built with Pygame.  

[ CONTROLS ]  
LMB -> Paint  
RMB -> Erase  
SCROLL -> Zoom  
MMB or LMB + SPACE -> Move  
TAB -> Toggle grid  
CTRL + R -> Clear canvas  
CTRL + S -> Save art  
ESC -> Quit
